# QuizApp 🎯

Eine interaktive Quiz-Anwendung, entwickelt mit **HTML**, **CSS** und **JavaScript**. Die App ermöglicht es Benutzern, Multiple-Choice-Fragen zu beantworten, ihren Fortschritt zu verfolgen und am Ende ihre Punktzahl zu sehen.

## 🚀 Features

- Multiple-Choice-Fragen mit vier Antwortmöglichkeiten
- Sofortiges Feedback bei richtiger oder falscher Antwort
- Soundeffekte für richtige und falsche Antworten
- Fortschrittsbalken während des Quiz
- Ergebnisanzeige nach Abschluss
- Quiz-Neustart per Klick
- Responsives Design

## 🛠️ Verwendete Technologien

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap
- Audio API

## ⚙️ Installation

Repository klonen:

```bash
git clone https://github.com/DEIN-USERNAME/quizapp.git
```

Projektverzeichnis öffnen:

```bash
cd quizapp
```

Abhängigkeiten installieren:

```bash
npm install
```

Anwendung starten:

```bash
npm start
```

Alternativ kann die Datei `index.html` direkt im Browser geöffnet werden.

## 🎮 Spielablauf

1. Quiz starten
2. Eine Antwort auswählen
3. Sofortiges Feedback erhalten
4. Mit „Next“ zur nächsten Frage wechseln
5. Nach der letzten Frage wird das Endergebnis angezeigt
6. Über „Restart“ kann das Quiz erneut gestartet werden
